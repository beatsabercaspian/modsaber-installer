# modsaber-installer
The new 2026 ModSaber Installer to relive the good ol' days.

This installer is a python script which fetches mod archives from the modsaber.org/archive urls from the Wayback Machine so you don't have to! Although it might be a bit slow it will definitely work. It supports the versions: 0.13.1, 0.13.0p1/0.13.0, 0.12.2, 0.12.0p1/0.12.0, and 0.11.2. (0.12.1 will get added soon!)

Anyway i hope you guys will enjoy playing such old versions of beat saber as i do, and also enjoy using the installer.
